## Description

Everyone's seen AI write code. Almost no one is shipping real software with it.

The gap between a clever demo and a production-grade product has a name: harness engineering — the discipline of designing the execution environment around AI agents so they can actually deliver work you'd put your name on.

In this 90-minute live session, Alexis and Exe from Southern Code will walk through how they've spent the last two years building real products with agentic coding workflows — Claude Code, MCPs, skills, hooks, custom tooling, automated QA, and the debugging patterns that keep long-running agents on track.

Built for founders, operators, and leaders at companies adopting AI coding — whether you write code or not. Live demo. Real questions. No slides about "the future of AI."
