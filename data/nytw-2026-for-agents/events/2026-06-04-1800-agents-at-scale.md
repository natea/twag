## Description

Demo agents are easy. Production agents are a different problem.

Join Modal, Braintrust, Deskree and Tavily for an evening of honest conversation about what it actually takes to ship AI agents that work reliably, at scale, and in real enterprise environments. No keynotes, no product pitches. Just practitioners from leading developer tools companies talking through the problems they've actually solved.

The centerpiece is a panel featuring engineers and builders from across the agent stack. They'll dig into the parts nobody talks about in blog posts: context management, orchestration failures, eval frameworks that actually work, observability at scale, and what "done" looks like when an agent is customer-facing.

This event is for:
Engineers building or maintaining agent-based systems
Technical leads evaluating agent infrastructure
Technical founders navigating the same production challenges
