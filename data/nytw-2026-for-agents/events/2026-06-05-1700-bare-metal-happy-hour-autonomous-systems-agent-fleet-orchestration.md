## Description

Are you an engineer or builder working on autonomous systems or agent fleet orchestration?

We're bringing together practitioners who are moving beyond individual AI tools and into autonomous, continuously running systems — whether you're maxing out managed API plans or running open source models on your own hardware.

Join us if you're building toward AI as a scalable workforce: agent loops that run for hours, parallelized workflows, and the infrastructure decisions that make or break production systems.

Expect to talk about things like:
↳ Scaling agent fleets on managed APIs vs. self-hosted inference
↳ Open source models and where local compute actually makes sense
↳ State persistence, monitoring, and reliability for long-running agents
↳ The real tradeoffs between cost, latency, and control at scale

Whether you're orchestrating agents through Claude or running Llama on bare metal, come ready to talk implementation details with people who are building the same way you are.
