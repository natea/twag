## Description

​Join Tavily, LangChain, Nebius and J.P. Morgan for a hands-on session where you’ll learn how to build a real-time market research agent that combines structured financial data with live web intelligence.

What is J.P. Morgan Innovation Economy Banking?

​
​What is Tavily?

​Tavily is a purpose-built search API for AI agents. It provides fast, structured, and relevance-optimized access to real-time web data, enabling developers to build agents that retrieve high-quality, up-to-date information with minimal noise.

What is Nebius?

Nebius is a purpose-built AI cloud platform designed for training, fine-tuning, and deploying modern AI models at scale. It provides high-performance GPU infrastructure, managed AI services, and developer-friendly tooling that enable teams to build and run AI workloads efficiently, reliably, and with full scalability.

What is LangChain?

LangChain is a purpose-built framework for developing applications powered by large language models (LLMs). It provides modular tools for orchestrating prompts, connecting models to external data sources, managing memory, and building AI agents, enabling developers to create intelligent, context-aware applications with minimal complexity.
