## Description

More and more, the primary user of a product isn't a person — it's an LLM reading docs, calling tools, and doing work on someone's behalf. The human has moved one layer up.

Zo and Tanagram are building into this shift from opposite ends. Zo gives individuals a personal AI cloud computer where agents act on their behalf. Tanagram shapes what coding agents produce inside engineering teams, turning tribal knowledge into guardrails the agent follows.

Join the founders for a candid conversation on designing for a user that reads everything, hallucinates confidently, and never sleeps — what reliably works, what reliably breaks, and how product thinking changes when tool schemas are the new UI.
