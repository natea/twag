## Description

This is an opportunity to get hands-on experience creating demos for companies using the latest features of ADK. Participants will be the very first to try Agent Development Kit (ADK) 2.0 in a 3.5-hour collaborative build session, giving you a unique opportunity to test its capabilities. The event centers on exploring brand-new features, specifically graph-powered orchestration and task-based agentic tools. Builders are invited to choose their own path to solve real-world customer use cases by:

- Migrating an existing agent system from a competitor's framework over to ADK 2.0.
- Refactoring a current ADK agent to take advantage of the new graph-based routing.
- Building an entirely new multi-agent system from scratch.


This event is a part of #BOSTechWeek—a week of events hosted by VCs and startups to bring together the tech ecosystem. Learn more at www.tech-week.com.
