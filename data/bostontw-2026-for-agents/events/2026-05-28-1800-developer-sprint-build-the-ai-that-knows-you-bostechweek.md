## Description

What does it mean to build AI that actually knows you?

This hackathon is about that shift, from reactive tools to adaptive systems. AI that remembers, learns, and evolves with you.

You’ll build using OpenClaw + Hermes Agent to prototype real applications for moccet. Think memory layers, proactive agents, and systems that go beyond prompts.

The format is simple:
- quick onboarding
– 3-hour build sprint
– live demos

Top teams get moccet credits + a path to keep building.

Expect a strong builder crowd from Harvard, MIT, and across Boston.
