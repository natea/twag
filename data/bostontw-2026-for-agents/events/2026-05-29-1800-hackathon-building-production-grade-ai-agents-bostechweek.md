## Description

Most AI agent demos fall apart the moment they leave the notebook. This hackathon is about the other kind — agents you'd actually ship. 

We'll go hands-on with a curated stack for building agents the way real software gets built: 
- observable  
- testable 
- safe

This is for programmers who've built a prototype agent  and want to know what "good" looks like.  You'll leave with:  
- A working agent you built yourself, wired up with tracing and evals  
- A concrete mental model for production agent architecture 
- Patterns you can apply for the rest of your career

This event is a part of #BOSTechWeek—a week of events hosted by VCs and startups to bring together the tech ecosystem. Learn more at www.tech-week.com.
